JSON (JavaScript Object Notation)

JSON (JavaScript Object Notation) เป็นรูปแบบของข้อมูลที่ใช้ในการแลกเปลี่ยนข้อมูลระหว่างเซิร์ฟเวอร์และไคลเอนต์ โดยมีรูปแบบที่เป็นข้อมูลแบบข้อความ (text-based) ที่สามารถอ่านได้ง่ายโดยมนุษย์และสามารถสร้างและแก้ไขได้ด้วยโปรแกรมคอมพิวเตอร์ได้อย่างสะดวก JSON มีโครงสร้างแบบกลางที่ประกอบด้วยคู่คีย์-ค่า (key-value pairs) ที่จัดเก็บข้อมูลในรูปแบบของออบเจ็กต์ (objects) หรืออาเรย์ (arrays) และสามารถซ้อนกันได้ โดยรูปแบบหลักของ JSON จะเป็นดังนี้: 

                        
                      {"key1": "value1",
                      "key2": "value2",
                      "key3": {"nestedKey1": "nestedValue1",
                               "nestedKey2": "nestedValue2"},                      
                      "key4": ["value3", "value4"]}
ในตัวอย่างข้างบนนี้ เป็น JSON ที่ประกอบด้วยออบเจ็กต์ที่มีคู่คีย์-ค่า และอาเรย์ โดยมีโครงสร้างดังนี้: "key1", "key2", "key3", "key4" เป็นคีย์ที่มีค่าเป็นข้อมูลชนิดต่าง ๆ "value1", "value2" เป็นค่าของคีย์ "key1" และ "key2" "key3" เป็นคีย์ที่มีค่าเป็นออบเจ็กต์ซ้อนอยู่ข้างใน "nestedKey1", "nestedKey2" เป็นคีย์ของออบเจ็กต์ซ้อน "key3" "key4" เป็นคีย์ที่มีค่าเป็นอาเรย์ของข้อมูล JSON มักถูกใช้ในการส่งข้อมูลระหว่างเว็บเซิร์ฟเวอร์และไคลเอนต์ผ่าน HTTP Request และ Response และยังถูกใช้ในการเก็บข้อมูลและการกำหนดค่าในการตั้งค่าของแอปพลิเคชันต่าง ๆ อีกด้วย เนื่องจาก JSON เป็นรูปแบบข้อมูลที่ยืดหยุ่น และสามารถอ่านได้ง่าย ไม่ว่าจะเป็นภาษาโปรแกรมมิ่งใดก็สามารถใช้ได้ง่ายและสะดวก
