CSS (Cascading Style Sheets)

CSS (Cascading Style Sheets) เป็นภาษาสไตล์ที่ใช้ในการแสดงผลและจัดรูปแบบเว็บเพจ (หรือเอกสาร XML) บนเว็บไซต์ โดย CSS ช่วยให้ผู้พัฒนาเว็บสามารถกำหนดรูปแบบการแสดงผลของเนื้อหาได้ เช่น การกำหนดสีพื้นหลัง, ขนาดตัวอักษร, รูปแบบตาราง, การจัดหน้าเนื้อหา, และการให้ลักษณะต่างๆ ให้กับองค์ประกอบต่างๆ ของเว็บ เช่น หัวเรื่อง (headings), ข้อความ (text), ลิ้ง (links), ภาพ (images) เป็นต้น CSS ทำงานร่วมกับ HTML เพื่อกำหนดลักษณะและรูปแบบต่างๆ ของเว็บไซต์ โดยการเชื่อมโยง CSS กับ HTML จะทำผ่านการระบุลิงก์ไปยังไฟล์ CSS จาก HTML ซึ่งจะบอกเว็บเบราว์เซอร์ว่าใช้รูปแบบ CSS ไฟล์นั้นเพื่อแสดงผลเว็บไซต์ เรียกใช้ CSS จะช่วยให้การจัดรูปแบบเว็บไซต์เป็นระเบียบและมีความสวยงามมากยิ่งขึ้น และยังช่วยให้เว็บไซต์มีความสอดคล้องกับมาตรฐานการเว็บพัฒนา (Web development standards) อีกด้วย
