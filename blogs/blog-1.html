HTML 5(Hypertext Markup Language 5)

HTML 5 (Hypertext Markup Language 5) เป็นเวอร์ชันล่าสุดของภาษา HTML ซึ่งใช้สำหรับสร้างและแสดงเว็บเพจบนเว็บไซต์ โดย HTML 5 ได้รับการพัฒนาขึ้นเพื่อปรับปรุงความสามารถและประสิทธิภาพในการสร้างเว็บเพจให้มีประสิทธิภาพมากยิ่งขึ้น โดยมีความสามารถและลูกเล่นใหม่ๆ เช่น การรองรับวิดีโอและเสียงโดยไม่ต้องใช้ Flash, การสนับสนุนภาพและแอนิเมชัน 2D/3D โดยไม่ต้องใช้ปลั๊กอินภายนอก, การทำงานออฟไลน์ (offline capabilities) , และการปรับปรุงโค้ดเพื่อให้เว็บไซต์ทำงานได้ดียิ่งขึ้นบนอุปกรณ์มือถือและแท็บเล็ต นอกจากนี้ HTML 5 ยังมีการรองรับและปรับปรุงฟอร์แมตต่างๆ เช่น ฟอร์แมตวามเสมอ, ฟอร์แมตสำหรับเว็บแอปพลิเคชัน (Web Application) และการใช้งานง่ายขึ้นในการทำงานร่วมกับ CSS (Cascading Style Sheets) และ JavaScript โดย HTML 5 มีความสามารถที่สามารถใช้งานได้กับเว็บเบราว์เซอร์หลายรูปแบบโดยไม่ต้องมีการปรับแต่งโค้ดให้แตกต่างกัน ทำให้เป็นมาตรฐานที่ดีในการพัฒนาเว็บไซต์และแอปพลิเคชันบนโลกอินเทอร์เน็ตในปัจจุบั
