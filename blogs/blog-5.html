Node.js

Node.js เป็นแพลตฟอร์มที่ใช้ในการเรียกใช้โค้ด JavaScript ฝั่งเซิร์ฟเวอร์ (server-side) ซึ่งมีความแตกต่างกับ JavaScript ที่ใช้ในเว็บเบราว์เซอร์ (client-side) โดยปกติแล้ว โปรแกรม JavaScript จะถูกดำเนินการบนเว็บเบราว์เซอร์เท่านั้น แต่ Node.js ทำให้สามารถรัน JavaScript ได้ที่ฝั่งเซิร์ฟเวอร์ด้วย Node.js ถูกพัฒนาโดย Ryan Dahl และถูกเปิดตัวเป็นโอเพนซอร์สในปี 2009 ซึ่งมีจุดมุ่งหมายในการช่วยให้ผู้พัฒนาสามารถสร้างแอปพลิเคชันเว็บที่มีประสิทธิภาพสูง โดย Node.js มีระบบเหตุการณ์ (event-driven) และโมเดล I/O ที่ไม่บล็อก (non-blocking I/O) ซึ่งช่วยให้สามารถจัดการกับการทำงานแบบพร้อมกันได้หลายอย่างโดยไม่ต้องรอให้การทำงานเสร็จก่อน Node.js มีประโยชน์ในการพัฒนาเว็บเซิร์ฟเวอร์, แอปพลิเคชันเครือข่าย (network applications), และไฟล์เซิร์ฟเวอร์ (file servers) โดยเฉพาะในงานที่ต้องการการทำงานแบบ real-time หรือการประมวลผลแบบกระจาย (distributed processing) อย่างชัดเจน นอกจากนี้ Node.js ยังมีระบบชุดเครื่องมือที่สมบูรณ์และเสถียรที่ช่วยในการพัฒนาอย่างเช่น npm (Node Package Manager) ซึ่งเป็นเครื่องมือในการจัดการและแบ่งปันโมดูลของ Node.js และ Express.js ซึ่งเป็นเฟรมเวิร์กสำหรับพัฒนาเว็บเซิร์ฟเวอร์ด้วย Node.js ที่มีความนิยมอย่างแพร่หลาย
